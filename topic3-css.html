📖 Description

What is CSS?

CSS (Cascading Style Sheets) is the language used to control the visual presentation of HTML documents. While HTML provides structure and content, CSS determines how that content looks — including colours, typography, spacing, layouts, animations, and responsiveness across different screen sizes and devices.

CSS works through a system of selectors and declarations. A selector targets an HTML element (e.g., h1, .card, #header), and a declaration sets a property and value (e.g., color: red;). The "cascading" nature means styles can be inherited and overridden according to a specific priority order called specificity.

CSS3 introduced powerful modern features: Flexbox and Grid for sophisticated, responsive layouts; @media queries for adapting designs to different screen widths; :hover and :focus pseudo-classes for interactive states; CSS custom properties (--variable-name) for reusable values; and @keyframes animations to bring pages to life without JavaScript.

In a real-world student dashboard, CSS is what transforms a plain HTML skeleton into a polished, professional interface — with colour-coded course cards, a responsive two-column grid layout, smooth hover transitions, and a sticky header. Good CSS is the difference between a bare, unusable page and a product people enjoy using.

/* ── CSS Custom Properties (Variables) ── */
:root {
  --primary:    #4a90e2;
  --secondary:  #7b68ee;
  --success:    #28a745;
  --bg:         #f4f6fb;
  --card-bg:    #ffffff;
  --text:       #2c3e50;
  --text-muted: #6c757d;
  --border:     #e1e8f0;
  --shadow:     0 4px 20px rgba(0, 0, 0, 0.08);
  --radius:     12px;
  --transition: 0.3s ease;
}

/* ── Reset & Base ── */
*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

body {
  font-family: 'Segoe UI', Tahoma, Geneva, sans-serif;
  background-color: var(--bg);
  color: var(--text);
  line-height: 1.6;
}

/* ── Sticky Header with Gradient ── */
header {
  background: linear-gradient(135deg, var(--primary), var(--secondary));
  color: white;
  padding: 1.5rem 2rem;
  display: flex;
  align-items: center;
  justify-content: space-between;
  box-shadow: var(--shadow);
  position: sticky;
  top: 0;
  z-index: 50;
}

header h1 { font-size: 1.5rem; font-weight: 700; }

/* ── Navigation ── */
nav ul { list-style: none; display: flex; gap: 0.5rem; }

nav ul li a {
  color: rgba(255, 255, 255, 0.85);
  text-decoration: none;
  padding: 0.4rem 1rem;
  border-radius: 20px;
  font-size: 0.9rem;
  transition: background var(--transition), color var(--transition);
}

nav ul li a:hover {
  background: rgba(255, 255, 255, 0.22);
  color: white;
}

/* ── Main Layout – CSS Grid ── */
main {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 2rem;
  max-width: 1100px;
  margin: 2rem auto;
  padding: 0 1.5rem;
}

/* Grades section spans full width */
#grades { grid-column: 1 / -1; }

/* ── Section Headings ── */
section h2 {
  font-size: 1.25rem;
  font-weight: 700;
  margin-bottom: 1.25rem;
  padding-bottom: 0.5rem;
  border-bottom: 3px solid var(--primary);
}

/* ── Course Cards with Hover Lift ── */
.course-card {
  background: var(--card-bg);
  border-radius: var(--radius);
  padding: 1.25rem 1.5rem;
  margin-bottom: 1rem;
  box-shadow: var(--shadow);
  border-left: 5px solid var(--primary);
  transition: transform var(--transition), box-shadow var(--transition);
}

.course-card:hover {
  transform: translateY(-5px);
  box-shadow: 0 10px 30px rgba(0, 0, 0, 0.13);
}

.course-card:nth-child(2) { border-left-color: var(--secondary); }

/* ── Grades Table ── */
table {
  width: 100%;
  border-collapse: collapse;
  background: var(--card-bg);
  border-radius: var(--radius);
  overflow: hidden;
  box-shadow: var(--shadow);
}

thead {
  background: linear-gradient(90deg, var(--primary), var(--secondary));
  color: white;
}

thead th { padding: 0.85rem 1rem; text-align: left; font-size: 0.88rem; }

tbody tr {
  border-bottom: 1px solid var(--border);
  transition: background var(--transition);
}

tbody tr:hover { background: #f0f4ff; }
tbody td { padding: 0.85rem 1rem; font-size: 0.9rem; }
tbody td:last-child { font-weight: 700; color: var(--success); }

/* ── Responsive Design ── */
@media (max-width: 768px) {
  main { grid-template-columns: 1fr; }
  #grades { grid-column: 1; }
  header { flex-direction: column; gap: 1rem; }
}

/* ── Footer ── */
footer {
  text-align: center;
  padding: 2rem;
  background: var(--text);
  color: rgba(255, 255, 255, 0.55);
  font-size: 0.85rem;
  margin-top: 3rem;
}
