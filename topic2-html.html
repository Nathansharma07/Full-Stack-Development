What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It is the backbone of every web page — defining what content appears and how it is organised using a system of elements represented by tags enclosed in angle brackets, such as <h1>, <p>, and <div>.

HTML5, the current standard, introduced semantic elements like <header>, <nav>, <main>, <section>, <article>, and <footer>. These elements carry meaning about the type of content they contain, making websites more accessible to screen readers and easier to index by search engines (SEO).

HTML works by nesting elements inside one another in a tree-like structure called the DOM (Document Object Model). The browser reads this tree and renders the visual output the user sees. Every web page begins with a <!DOCTYPE html> declaration, followed by an <html> element containing a <head> (metadata) and a <body> (visible content).

In a real-world scenario, HTML is used to build the structural skeleton of every web page — from a simple blog article to a complex student learning portal. Without HTML, there would be no content for CSS to style or for JavaScript to make interactive. It is the foundation upon which the entire web is built.
