What is Bootstrap?

Bootstrap is the world's most popular open-source front-end CSS framework, originally created by developers at Twitter. It provides a comprehensive collection of pre-built responsive CSS classes, UI components, and JavaScript plugins that dramatically accelerate web development — allowing developers to build consistent, mobile-first layouts without writing large amounts of custom CSS from scratch.

Bootstrap's core feature is its 12-column responsive grid system. Using classes such as col-md-6 or col-lg-4, developers can define how content is distributed across different screen widths — automatically adapting layouts for mobile, tablet, and desktop with no manual media queries.

Bootstrap also ships with a rich set of ready-made components: Navbars (collapsible, responsive navigation bars), Cards (flexible content containers), Badges (status indicators), Progress Bars, Modals, Accordions, Dropdowns, Alerts, and many more. These follow consistent design patterns and accessibility standards out of the box.

Bootstrap's utility classes — such as mt-3, text-center, d-flex, and justify-content-between — allow precise control over spacing, alignment, and display directly in HTML. This approach is the same philosophy adopted by modern utility-first frameworks like Tailwind CSS. Bootstrap is used by millions of websites worldwide and is an essential tool in any front-end developer's toolkit.
